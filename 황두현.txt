main이란?
: java로 구현된 프로그램이 시작될 때 가장 먼저 실행되는 메서드.

주석이란?
: 프로그램을 수행하는 데 영향을 끼치지 않는 설명 또는 의견을 적은 글.
1. 행 주석: //로 시작해서 행의 끝까지가 주석문이 됨.
2. 범위 주석: /*로 시작해서 */로 끝남. 여러 줄의 문장을 주석 처리할 때 사용.

변수란?
: 데이터를 저장할 수 있는 메모리 공간. 저장된 값은 변경될 수 있음.

- '변수선언'에 대해 설명하세요.
: 변수라는 메모리 공간의 이름을 정의하는 것. 변수의 데이터타입의 크기만큼 메모리 공간을 확보한다.

- 변수를 선언할 때 '변수이름'을 작성하는 방법에 대해 설명하세요.
: 변수의 이름은 영문자(대소문자), 숫자, 언더바(_), 달러($)로만 구성
변수의 이름은 숫자로 시작할 수 없음
변수의 이름 사이에는 공백을 포함할 수 없음


- 변수를 선언할 때 사용할 'Datatype'의 종류에 대해 알아보고, 설명하세요.
1. 기본형(자료형): 논리형(Boolean), 문자형(char), 정수형(short, byte, int, long), 실수형(float, double)으로 구성
2. 참조형

예약어란?
:  프로그래밍 언어에서 미리 사용하기로 약속한 단어. 식별자로 사용할 수 없는 단어를 뜻함.
매번 긴 코드를 사용할 수 없으니 함수로 정의해놓은 것.
Ex) print, printf, int, char, for, while, if, ...

코드블록이란?
:
코드 블록은 시각적 스크립팅 환경 내의 문자 스크립팅 인터페이스이다.
프로그래밍 코드를 삽입할 경우 사용하며, 삽입된 코드는 보기 좋게(Syntax Highlighting)표시 된다.
숫자, 문자열, 수식 및 기타 데이터 유형으로 사용할 수 있다.
코드 블록은 사용자가 직접 만들어 가는 도구용으로 설계되었으므로 사용자는 코드 블록에서 임의의 변수를 정의할 수 있으며,
	이러한 변수는 노드의 입력에 자동으로 추가됩니다.
코드 블록을 사용하면 사용자가 입력 지정 방식을 유연하게 결정할 수 있습니다.

시각적 스크립팅 환경 내의 문자 스크립팅 인터페이스다. 
숫자, 문자열, 수식 및 기타 데이터 유형으로 사용할 수 있다. 
코드 블록은 Dynamo용으로 설계되었으므로 사용자는 코드 블록에서 임의의 변수를 정의할 수 있으며,
	이러한 변수는 노드의 입력에 자동으로 추가된다
